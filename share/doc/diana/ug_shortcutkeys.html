List of shortcut keys (key combinations)

Editing commands only work in edit mode. Apart from that, all commands work in both edit mode and normal mode.

Graphics window commands
Zoom in z
Zoom out shift-z or x
Selected map section F2
Define map section shift-F2
Previous map section F3
Next map section F4
Big map section F5
Medium map section F6
Small map section F7
Detailed map section F8
Add to quick menu F9
Previous plot F10
Next plot F11
Move area to the right right arrow
Move area to the left left arrow
Move area up up arrow
Move area down down arrow
Reverse arrow keys r
Switch between quick menus ALT+arrow keys
Standard area Home
Apply Ctrl-u
Increase/decrease size of station icons (i.e. modfly icons) Ctrl +/-

Animation commands
One time step forward Ctrl-right arrow
One time step backwards Ctrl-left arrow
Change the time step interval Ctrl-up/down arrow
Animate forwards Ctrl+Shift-right arrow
Animate backwards Ctrl+Shift-left arrow
Stop animation Ctrl+Shift-down arrow
Loop Ctrl+Shift-up arrow

Level and class/type commands
Level up Ctrl-page-up
Level down Ctrl-page-down
Class/type (EPS cluster,run,...) up Shift-page-up
Class/type (EPS cluster,run,...) down Shift-page-down

File commands
Print Ctrl-p
Quit Ctrl-q

Menu commands
Hide all Ctrl-d
Quick menu F12
Map dialogue Alt-k
Field dialogue Alt-f
Observation dialogue Alt-o
Satellite dialogue Alt-s
Edit dialogue Alt-e
Object dialogue Alt-j
Trajectory dialogue Alt-t
Vertical profiles Alt-v
Vertical cross section Alt-c
Help F1

Observation commands
Draw more observations page-up/page-down

Edit commands
Draw new object n
End drawing p
Delete points Delete
Turn front v
Change object type t/Shift-t
New points on front/area f
Split front k
Connect front j
Join two fronts of the same type e
Disconnect fronts q
Increase/decrease symbol size +-
Standard symbol size b/Shift-b
Mark objects m
Cancel mark u/c
Hide objects g
Observations one time step forward Shift-right arrow
Observations one time step backward Shift-left arrow
Hide boundary line l
Show a region analysis 1,2,3..
Undo edit of field or object Ctrl-z
Redo edit of field or object Ctrl-y
Save edit Ctrl-s
Copy object Ctrl-c
Paste object Ctrl-v
Change symbol text Ctrl-e
Rotate sigmap symbol Ctrl+/-
Show/hide white box (sigmap symbol) o

Vertical profile and wave spectrum commands
One time step forward Ctrl-right arrow
One time step backward Ctrl-left arrow
Change station Ctrl-up/down arrow

Vertical cross section commands
Zoom in z
Zoom out shift-z or x
Move area to the right right arrow
Move area to the left left arrow
Move area up up arrow
Move area down down arrow
Reverse arrow keys r
Standard area Home
One time step forward Ctrl-right arrow
One time step backward Ctrl-left arrow
Change vertical cross section Ctrl-up/down arrow

Commands for on/off buttons
Focus on the buttons End
Move between the buttons right/left arrow
Turn on/off button up/down arrow
