Profet drawing tools

The drawing tool is available when creating a new object or when editing an existing object. The tool is based on a set of drawing modes shown in a seperate toolbar. The toolbar is shown in the main window.
  1. Select Object - Mode for selecting objects by clicking on the map.
  2. Draw Area - Mode for drawing and redrawing areas.
  3. Add Area - Mode for adding another area.
  4. Cut Area - Mode for removing another area.
  5. Move Area - Mode for moving area to another location.
  6. Add Point - Mode for adding a point (vertex) between two existing points.
  7. Remove Point - Mode for removing existing point from area.
  8. Move Point - Mode for move a point to another location.
  9. Spatial Interpolation - Mode for interpolating the area.
  10. Undo - Undo last action
  11. Redo - Redo last action
You can change mode by clicking in the toolbar or by using shortcuts (1-9). The shortcuts only works when the main window is in focus.

Draw an area
  1. Select draw mode from the toolbar (shortcut 2).
    • Your mouse pointer should show a pencil on the map.
    • Existing area will be overwritten when you start to draw.
  2. Draw by clicking with the left mouse button in the map.
    • Add individual points by clicking.
    • Add points continuously by holding the left mouse key down while moving the pointer.
  3. Finish drawing by clicking the right mouse button.
    • Rightclick adds the last point in your area.
    • The area will be closed even if not the last point is close to the first point.
  4. The finished area will be filled.
    • Moving mode will be selected automatically. This is to prevent users from overwriting the area next time clicking on the map.
    • The area will be adjusted to the underlying grid and might change.
    • Points that do not change the shape of the finished area will be removed.
    • The mask (grid points that will be affected by the algorithm) is shown in dark color.

Add area
  1. Select add mode from the toolbar (shortcut 3).
  2. Draw an area in the same way as in draw mode.
    • The area must be partly overlapping with the existing area.
  3. Finish drawing by clicking the right mouse button.
  4. The area will include the new selection.
    • Note that the new area will be adjusted to the underlying grid. If you have changed something that is less than the grid size, it will not change.

With continuous drawing the operation will not always work.

Cut area
  1. Select cut mode from the toolbar (shortcut 4).
  2. Draw an area in the same way as in draw mode.
    • The area must be partly overlapping with the existing area.
  3. Finish drawing by clicking the right mouse button.
  4. The new selection will be removed from the area.
    • Note that the new area will be adjusted to the underlying grid. If you have changed something that is less than the grid size, it will not change.

With continuous drawing the operation will not always work.

Move area
  1. Select move mode from the toolbar (shorcut 5).
  2. Click and hold the left mouse button.
    • Only the outline of the area is visible.
  3. Move the outline to the desired position.
  4. Release the left mouse button to place the area.
    • The area will now be displayed as normal.
    • The area will be moved to the closest grid point.

Add point
  1. Select the add point mode from the toolbar (shortcut 6).
    • The existing points will be marked with squares.
  2. Move the cursor to the line where you want to add a point.
    • The line closest to the mouse pointer will be split by the point you are pointing at.
    • The old line is shown as a dotted line.
  3. Press the left mouse button to add the point you are pointing at.
  4. The new point will be added and the mode is set to move point.
  5. To add more points you must select the mode again. It is convenient to use hotkey to add many points.

Remove point
  1. Select remove point mode from the toolbar (shortcut 7).
    • The existing points will be marked with squares.
  2. Move the cursor close to the point you want to remove.
    • The point closest to the mouse pointer will be highlighted.
  3. Press the left mouse button to remove the point.

Move point
  1. Select move point mode from the toolbar (hotkey 8).
    • The existing points will be marked with squares.
  2. Move the mouse pointer close to the point you want to move.
    • The point closest to the mouse pointer will be highlighted.
  3. Press and hold the left mouse button.
    • You will now see only the outline of the area.
  4. Move the mouse pointer while you hold the mouse button down.
  5. Release the mouse button where you want to set point.
  6. The new area will be displayed as normal.

Undo / Redo
  1. You can undo the previous change by pressing the Undo button.
  2. The previous area will then appear on the map.
  3. To redo, press the redo button on the right.
