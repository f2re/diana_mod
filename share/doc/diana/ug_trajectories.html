Trajectories

  • Colour - changes the colour of the trajectories.
  • Line width - changes the line width of the trajectories.
  • Line type - changes the line type of the trajectories.
  • Time marks - for marking chosen time interval for the trajectories.
  • Number of positions - every start position can either be just one point or one point with four or eight positions around it.
  • Radius - distance from the centre point to the four or eight surrounding points.
  • Select positions on the map - When this is selected, the start position can be selected by clicking the map.
  • Start positions can also be typed in, latitude then longitude. The coordinates can be entered either degrees:minutes:seconds or decimal degrees and width either +/- in front or NESW after.
  • The list shows selected start positions. Even though five or nine markers are selected, only the coordinates of the centre point is shown
  • Delete - deletes marked position.
  • Delete all - deletes all start positions.
  • Start computation - looks among displayed fields and finds the first which can be used. The name of the field is written in the dialogue. In this case "HIRLAM.00 VIND 700hPa". If no field is found, "No fields" is shown. The computation starts when the time is changed. The time is changed by either using the animation buttons or with the short commands Ctrl-left/right arrow. If the time slider is used, be aware that only the first and last field is used in the computations. Also note the spinbox which shows the time steps, all of the fields will not be included in the computation if the selected time step is too large.
  • Quit - Deletes all trajectories and start positions.
