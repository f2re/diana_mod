Maps

  • Use the map dialogue to choose area, projection and map.
  • One Area w/projection is chosen from the top list.
    • Model/sat-area means that the field or satellite picture defines the projection and area.
    • The rest of the list is from the setup-file and defines regular cut-outs with projection. The names should be self-explanatory.
  • Choose background colour from the list - applies to all chosen maps.
  • Lat/lon lines turn on/off lines, with options for colour, thickness, density (decimal degrees) and plot position.
  • Show frame puts a frame around the chosen area (in the top list). This can be turned on/off, with options for colour, line thickness and plot position. Note that this will not be shown when model/sat-area has been chosen.
  • Note that it is possible to turn off all maps and only use the background colour optionally including lat/lon lines and area frame.
  • Choose one or more maps. Maps can be unselected by clicking them a second time in the list, or by using the Delete-button on an element in the list Selected maps. Maps with names that contain auto have support for different resolutions depending on the size of the area being shown.
  • For each map there are choices for contour lines. These are: colour, line thickness and position in relation to the other elements on the map (highest, lowest, auto). The options apply to the map highlighted in Selected maps.
  • For some maps Filled land can be chosen and colour and position is set. For these, contour lines can be turned off.
  • All selections are logged to file - so they are kept for the next time Diana is run. There are exceptions for high-resolution maps which would be very hard to run for a large area.
  • Use Save favourite when you want to save a map-layout you like. This is easily retrieved with Use favourite
Definition of areas/projections and maps are in the MAP_AREA and the MAP_TYPE sections in the setup file. List of plot settings for maps for use in quick menues or in batch is found in the batch-section.
