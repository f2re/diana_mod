Objects

The object dialogue is used to open object files from earlier analyses or prognoses.


  • First the type of object file is selected, e.g. met.no surface analysis.
  • The time can be selected in three ways.
    • Auto - the time from the time slider is used.
    • Time - all times are shown and can be selected from.
    • File - filenames are shown and can be selected from.
    When "Time" or "File" is selected a new file can not be selected with the time slider or the animation-buttons.
  • Selected files show what is actually selected.
  • Types of objects displayed is selected by clicking in fronts, symbols, areas, form.
  • Delete- button, deletes all selections.
  • Refresh - the file list is reread from the hard drive.
  • Time diff - maximum allowed time difference between chosen time and actual time of the object file. The file closest in time is always used.
  • Alpha - transparency of objects to be displayed. If alpha is less than 1.0, the objects will be transparent and easily separated from objects which are being edited.
  • Comments - show comments.
