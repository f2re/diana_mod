Observations

  • On the top line plot type is selected.
  • The next lines contain data types. Multiple types can be selected.
  • Then select the parameters which are to be plotted by either manually select each parameter or buy pressing the All, None or Default button.
  • Show all positions - displays an x where there is no room to display the observation. By clicking the x, the observation is displayed. PageUp and PageDown are used to draw the observations which there are no room for in the first picture.
  • Temp. as integers - shows the temperature, max/min temperature, dew point and sea temperature as integers (no decimals).
  • All observations (mixing different times)- displays all observations within the given time interval. This only applies to synoptic data, non-synoptic data are always displayed. The observations are sorted so that the ones which fit best time-wise are displayed first.
  • PPPP - MSLP-field - during editing shows the difference between observed pressure and the MSLP-field in the observation point. For synop plots the difference is shown instead of the observed pressure.
  • Selected observation in popup window - when pressing an observation point in the main window, a pop up window shows the observation according to what is specified in the setup file. If there is no specification in setup, all parameters selected will be shown.
  • Positions only - chosen marker is shown in all observation points.
  • Criteria - chosen criteria are used. Parameters which have critera are marked with a different colour. Click "More >>" to edit the criteria.
  • For the plot type "Pressure", pressure level needs to be selected. Either a particular level or the same as the field. (In practice the first field displayed.)
  • Density - how close the observations are displayed.
  • Size - size of the observations.
  • Timediff - maximum allowed deviation between chosen time and observation time. For synoptic data the file valid at the selected time is used, while for non-synoptic data all files with data within the selected time interval are read. The files validity time is set in the setup-file.
  • A priority list contains the name of preferred stations. The first station in the list is displayed first, thereafter as many as there is room for. Stations not in the list are also displayed if there are room. Priority lists are defined in the setup file.
  • Show only prioritized - If checked; show only the stations contained in the chosen priority list.
  • Sort - If selected; sort the observation plots accordning to the value of the selected parameter. The sort order is selected by pressing either the Asc och the Desc radio buttons.
  • Refresh - the list of files is reread.
  • Show all - Displays all plot types with the selected data types if clicked. When not clicked, only the selected plot types are plotted.
  • The - button on the Status line updates the displayed observations if the relevant data files have changed since last read.

Criteria

Criteria can be given for what stations to show, depending on the value of individual parameters. The parameters or the observation can be given a colour, depending on value.

  • List of criteria - Choose a list of criteria. The lists can be added in the setup-file or made in the dialogue. All saved changes are logged.
  • Criteria - shows the criteria in the selected list. Selected criteria can be altered or deleted. Add a new criterion by right-clicking the parameter.
  • Delete - deletes selected criteria.
  • Delete all - deletes all criteria in the selected list.
  • Plot - the observation matching the criteria are plotted
    • Colour - parameter - parameters matching the criteria are plotted in the selected colour.
    • Colour - whole synop - observations matching the criteria are plotted in the selected colour.
    • Marker - observations matching the criteria are marked with the selected marker below.
  • Colour - Choose colour, only applies if "Colour" has been selected above.
  • Marker - Choose marker, only applies if "Marker" has been selected above.
  • Limit - Select operator and limit value. If no operator is selected and the parameter has been observed it will be plotted if "Plot" has been selected, while if "Colour" has been selected, it will be plotted in the selected colour. The precision of the limit value can be adjusted.
  • Save criteria list - enter filename and click "Save".
List of data types, data files etc. are in the OBSERVATION_FILES section in the setup file. Predefined criterias are found in the OBSERVATION_CRITERIA section. What syntax to use in setup file for "Selected observation in popup window" can be found in the OBSERVATION_POPUP_SPEC section. List of plot settings for observations for use in quick menues or in batch is found in the batch-section.
