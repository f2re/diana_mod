Menu bar

The menu bar contains four menu items File, Options, Show and Help.

  • File:
    • Save Picture Saves the contents to file. Files can be saved as png, xpm, bmp or eps. The first three saves a screen dump, while eps (encapsulated postscript) gives a full postscript-product. The suffix used for the filename decides what format the file is saved as. Default is png.
    • Email Picture Saves the screen dump to a temporary file as png, asks for email details, and sends the picture via email.

    • Save animation Save animation from current fields, satellite images, etc., using current settings. MPEG2 (*.mpg/*.mpeg) and MSMPEG4V2 (*.avi) is so far the only supported codecs, and this will probably not change if not requested as a new feature/enhancement (through http://diana.bugs.met.no).

      To save an animation, a fixed time interval must first be set, using the time limiting button (see the "Animation" section, under "Time control and animation" in the index).

      NOTE 1: It is not possible to use Diana while making an animation.
      NOTE 2: Compile Diana with VIDEO_EXPORT to enable this feature.

    • Print Prints the contents of the graphics window.
    • Quit Ends Diana
  • Options:
    • X,Y positions Swaps between showing geographical and map-coordinates in the status bar.
    • Speed buttons Adds a button in the status bar for every element (field, observation, radar, etc.) on the map. These are used for quickly turning on/off visibility for each element.
    • Archive mode Gives access to archived files with fields, observations and objects.
    • Automatic element choice Automatically selects elements (vertical profiles, vertical cross sections, observations, etc.) when the mouse is pointing at them. If this is off, the elements have to be clicked.
    • Select Font Used for changing the font used in all dialogues.
  • Show: This menu allows for displaying/hiding the different dialogues.
    • Apply plot activates selected changes
    • Add to quickmenu (F9) Lets you add a plot to your own quickmenu. See also Quick menus.
    • Previous plot and Next p lot lets you navigate a list of previously performed plots. The shortcut keys for these are F10 and F11, you can also use ALT+arrow keys to navigate the history.
  • Help:
    • Documentation This document.
    • News Shows news since you last ran Diana.
    • About Diana Shows information about version etc.

Toolbars

The main window has two toolbars, Dataaccess/Tools and Time control.

The buttons on Data/Tools show/hides the dialogues for the available data in the system and tools for editing/trajectory-calculations etc. The button to the top/left applies the selections made in the dialogues.

Map-, field-, observation-, satellite-, and the object dialogue have the buttons "Apply" and "Hide" in the bottom.

  • "Apply" shows what is currently chosen in all the dialogues.
  • "Hide" hides the dialogue (same as clicking the icon/button on the toolbar.
The Apply-button on the toolbar (the thumb) resets the area, while the Apply-button in the dialogues and the shortcut Ctrl-U keeps the current area/last shown area.

The Time control toolbar has arrows to go backwards and forwards in time, and a time slider that shows the current time in the plotting window. The arrows to the right change the level for the fields.
